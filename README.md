## Installation (Manual)

1. Download this repo from releases
   
2. Go into your files and double-click the .zip file you just downloaded. It will create a folder
   
3. Go to your Chrome browser and enter "chrome://extensions" in the search bar. At the top right corner there will be a toggle for developer mode. Make sure that toggle is on
   
4. On the left of your screen, there will be "Load Unpacked" button. Click that button, and it will open up your file menu. Select the folder you created in Step 2
   
5. The extension will now pop up in your All Extensions section. Just make sure you pin the extension (details -> Pin to toolbar) for easy access.



## Usage
Go to your Aeries *gradebook* page to test it out. Follow the directions on the extension when you click it. The calculator button to the right on the extension popup enables what-if mode.
